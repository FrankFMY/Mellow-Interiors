# Mellow Interiors

Landing page for an interior design studio. Refined aesthetic with sage green palette, Inter + Playfair Display typography, smooth entrance animations.

## Tech Stack

- **Next.js 16** — App Router, Turbopack
- **Tailwind CSS v4** — `@theme` directive for custom design tokens
- **Framer Motion** — Scroll-triggered animations
- **Lucide React** — Icon system

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
app/components/
  Header.tsx      — Fixed navbar with mobile hamburger menu
  Hero.tsx        — Full-screen hero with studio tagline
  Portfolio.tsx   — Project gallery showcase
  Process.tsx     — Design process steps
  About.tsx       — Studio story and team
  Contact.tsx     — Contact form section
  Footer.tsx      — Site footer with navigation
```
